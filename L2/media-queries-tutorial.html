Media Queries Tutorial

What are Media Queries?

Media queries are a CSS technique that allows you to apply different styles based on various conditions like screen size, device orientation, or resolution. They are essential for creating responsive designs that work across different devices.

Example 1: Basic Media Query

This example demonstrates a simple media query that changes the background color when the screen width is 768px or less.

This box changes color on smaller screens!

@media screen and (max-width: 768px) { .basic-example { background-color: #e0e0e0; } }

Common Media Query Breakpoints

/* Mobile devices */ @media screen and (max-width: 480px) { } /* Tablets */ @media screen and (min-width: 481px) and (max-width: 768px) { } /* Small laptops */ @media screen and (min-width: 769px) and (max-width: 1024px) { } /* Desktops */ @media screen and (min-width: 1025px) { }

Best Practices

  • Always use a mobile-first approach when possible
  • Test your media queries across different devices
  • Use relative units (rem, em) instead of pixels when possible
  • Keep your media queries organized and well-commented
  • Consider using CSS variables for consistent values across breakpoints
