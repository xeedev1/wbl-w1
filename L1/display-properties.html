Block Elements:

  • Take up full width by default
  • Start on a new line
  • Can have width and height set
  • Can have margins and padding on all sides
  • Common block elements: div, p, h1-h6, section, article

Inline Elements

Inline Element 1 Inline Element 2 Inline Element 3

Inline Elements:

  • Only take up as much width as necessary
  • Don't start on a new line
  • Cannot have width and height set
  • Only horizontal margins and padding work
  • Common inline elements: span, a, strong, em

Inline-block Elements

Inline-block 1
Inline-block 2
Inline-block 3

Inline-block Elements:

  • Flow like inline elements
  • Can have width and height set
  • Can have margins and padding on all sides
  • Don't start on a new line
  • Common use: navigation items, buttons, images

Flex Container:

  • Creates a flexible box layout
  • Items can grow and shrink
  • Items can be aligned and distributed
  • One-dimensional layout (row or column)
  • Common use: navigation bars, card layouts, forms

Grid Container:

  • Creates a two-dimensional grid layout
  • Items can be placed in specific grid areas
  • Supports both rows and columns
  • Can create complex layouts easily
  • Common use: page layouts, image galleries, dashboards

Note: This demo shows the basic behavior of different display properties. Each has its own use cases and can be combined with other CSS properties for more complex layouts.
